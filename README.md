# 🎮 坦克大战 (Tank Battle)

经典坦克大战游戏的 HTML/JavaScript 实现。

## 在线试玩

直接打开 `坦克大战.html` 即可游玩！

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 移动坦克 |
| 空格键 | 射击 |
| P | 暂停/继续 |

## 游戏特性

- 🧱 **砖墙** — 可被子弹摧毁
- ⚙️ **钢板** — 不可摧毁
- 🏠 **基地** — 保护你的总部！
- 🤖 **AI敌人** — 自动寻路、追踪和射击
- ❤️ **道具** — 修复、加速、强化火力
- 📊 **关卡系统** — 随关卡提升难度

## 项目结构

```
├── 坦克大战.html    # 单文件游戏（含所有HTML/CSS/JS）
├── README.md        # 本文件
```

## 技术栈

纯前端 HTML5 + Canvas + JavaScript，无需任何外部依赖。

打开即玩，适合部署到 GitHub Pages 或任何静态托管服务。
